# Targets API

`pnpm i`

## Running locally

```bash
pnpm dev
```

## Static analysis

```bash
pnpm check
```

## Building

Creates a production version of the app:

```bash
pnpm build
```

## Contributing

Always start your work in a new branch.

Run the following command as a last check before opening a PR

```bash
pnpm precommit
```
